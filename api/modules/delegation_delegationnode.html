External module Delegation/DelegationNode

Delegation nodes are used within the KILT protocol to construct the trust hierarchy.

Starting from the root node, entities can delegate the right to issue attestations to Claimers for a certain CTYPE and also delegate the right to attest and to delegate further nodes.

permissionsAsBitset

  • permissionsAsBitset(delegation: IDelegationNode): Uint8Array
    • Defined in delegation/DelegationNode.utils.ts:18

    Creates a bitset from the permissions in the array where each enum value is used to set the bit flag in the set.

    ATTEST has 0000000000000001 (decimal 1) DELEGATE has 0000000000000010 (decimal 2)

    Adding the enum values results in a decimal representation of the bitset.
