getBalance

  • getBalance(accountAddress: string): Promise<BN>
    • Defined in balance/Balance.chain.ts:85
    description

    Fetches the current balance of the account with [accountAddress]. Note that balance amount is in µ-Kilt and must be translated to Kilt-Coin

    example

    import * as sdk from '@kiltprotocol/prototype-sdk';

const address = ...
sdk.Balance.getBalance(address)
  .then((balance: BN) => {
    console.log(`balance is ${balance.toNumber()}`)
  })

    Parameters

    • accountAddress: string

      address of the account for which to get the balance.

    Returns Promise<BN>

    a promise containing the current balance of the account

listenToBalanceChanges

  • listenToBalanceChanges(accountAddress: string, listener?: undefined | function): Promise<BN>
    • Defined in balance/Balance.chain.ts:41
    description

    Attaches the given [listener] for balance changes on the account with [accountAddress]. Note that balance amount is in µ-Kilt and must be translated to Kilt-Coin

    example

    import * as sdk from '@kiltprotocol/prototype-sdk';

const address = ...
sdk.Balance.listenToBalanceChanges(address,
  (account: IPublicIdentity['address'], balance: BN, change: BN) => {
    console.log(`Balance has changed by ${change.toNumber()} to ${balance.toNumber()}`)
  });

    Parameters

    • accountAddress: string

      address of the account on which to listen for balance changes.

    • Optional listener: undefined | function

      listener to receive balance change updates

    Returns Promise<BN>

    a promise containing the current balance of the account
