  • countNodeDepth(attester: DidUri, attestation: IAttestation): Promise<number>
    • Defined in core/src/delegation/DelegationNode.utils.ts:46

    Traverses a delegation tree and counts the number of delegation nodes between an attestation and an ancestral delegation node owned by attester.

    throws

    SDKError If the attester is neither the owner nor in the delegation tree of attestation.

    Parameters

    • attester: DidUri

      Identity to be located in the delegation tree.

    • attestation: IAttestation

      Attestation whose delegation tree to search.

    Returns Promise<number>

    0 if attester is the owner of attestation, the number of delegation nodes traversed otherwise.

permissionsAsBitset

  • permissionsAsBitset(delegation: IDelegationNode): Uint8Array
    • Defined in core/src/delegation/DelegationNode.utils.ts:29

    Creates a bitset from the permissions in the array where each enum value is used to set the bit flag in the set.

    ATTEST has 0000000000000001 (decimal 1). DELEGATE has 0000000000000010 (decimal 2).

    Adding the enum values results in a decimal representation of the bitset.

    Parameters

    • delegation: IDelegationNode

      The delegation from which you want to convert the permissions to bitset.

    Returns Uint8Array

    The bitset as single value uint8 array.
