createLocalDemoFullDidFromSeed

  • createLocalDemoFullDidFromSeed(keystore: DemoKeystore, mnemonicOrHexSeed: string, generationOptions?: { encryptionKeyType?: NaclBox; endpoints?: ServiceEndpoints; keyRelationships?: Set<Omit<KeyRelationship, "authentication">>; signingKeyType?: SigningAlgorithms }): Promise<FullDidDetails>
    • Defined in did/src/DemoKeystore/DemoKeystore.utils.ts:84

    Creates an instance of FullDidDetails for local use, e.g., in testing. Will not work on-chain because identifiers are generated ad-hoc.

    Parameters

    • keystore: DemoKeystore

      The keystore to generate and store the DID private keys.

    • mnemonicOrHexSeed: string

      The mnemonic phrase or HEX seed for key generation.

    • generationOptions: { encryptionKeyType?: NaclBox; endpoints?: ServiceEndpoints; keyRelationships?: Set<Omit<KeyRelationship, "authentication">>; signingKeyType?: SigningAlgorithms } = {}

      The additional options for generation.

      • Optional encryptionKeyType?: NaclBox

        One of the supported EncryptionAlgorithms to generate the DID encryption keys.

      • Optional endpoints?: ServiceEndpoints

        The set of service endpoints that must be added to the DID.

      • Optional keyRelationships?: Set<Omit<KeyRelationship, "authentication">>

        The set of key relationships to indicate which keys must be added to the DID.

      • Optional signingKeyType?: SigningAlgorithms

        One of the supported SigningAlgorithms to generate the DID verification keys.

    Returns Promise<FullDidDetails>

    A promise resolving to a FullDidDetails object. The resulting object is NOT stored on chain.
