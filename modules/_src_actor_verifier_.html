    Parameters

      The Claimer's presentation of the Credentials that should be verified, the result of createPresentation.

    • session: IVerifierSession

      The Verifier's private verification session created in finalize.

    • Optional _latestAccumulators: Accumulator[]

      The list of the latest accumulators for each Attester which signed a Credential of this presentation.
