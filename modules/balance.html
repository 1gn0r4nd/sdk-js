getBalances

  • getBalances(accountAddress: IPublicIdentity["address"]): Promise<Balances>
    • Defined in packages/core/src/balance/Balance.chain.ts:41

    Fetches the current balances of the account with [accountAddress]. Note that the balance amounts are in Femto-Kilt (1e-15)and must be translated to Kilt-Coin.

    example

    
const address = ...
sdk.Balance.getBalances(address)
  .then((balanceData: AccountData) => {
    console.log(`free balances is ${balanceData.free.toNumber()}`)
  })

    Parameters

    • accountAddress: IPublicIdentity["address"]

      Address of the account for which to get the balances.

listenToBalanceChanges

  • listenToBalanceChanges(accountAddress: IPublicIdentity["address"], listener: (account: IPublicIdentity["address"], balances: Balances, changes: Balances) => void): Promise<UnsubscribePromise>
    • Defined in packages/core/src/balance/Balance.chain.ts:70

    Attaches the given [listener] for balance changes on the account with [accountAddress]. Note that the balance amounts are in Femto-Kilt (1e-15) and must be translated to Kilt-Coin.

    example

    const address = ...
const unsubscribe = await sdk.Balance.listenToBalanceChanges(address,
  (account: IPublicIdentity['address'], balances: Balances, changes: Balances) => {
    console.log(`Balance has changed by ${changes.free.toNumber()} to ${balances.free.toNumber()}`)
  });
// later
unsubscribe();

    Parameters

    • accountAddress: IPublicIdentity["address"]

      Address of the account on which to listen for all balance changes.

    • listener: (account: IPublicIdentity["address"], balances: Balances, changes: Balances) => void

      Listener to receive all balance change updates.

          • balances: Balances
          • changes: Balances

          Returns void

makeTransfer

  • makeTransfer(accountAddressTo: IPublicIdentity["address"], amount: BN, exponent?: number): Promise<SubmittableExtrinsic>
    • Defined in packages/core/src/balance/Balance.chain.ts:129

    Transfer Kilt [amount] tokens to [toAccountAddress]. Note that the value of the transferred currency and the balance amount reported by the chain is in Femto-Kilt (1e-15), and must be translated to Kilt-Coin.

    example

    const identity = ...
const address = ...
const amount: BN = new BN(42)
const blockchain = await sdk.getConnectionOrConnect()
sdk.Balance.makeTransfer(address, amount, 0) //
  .then(tx => BlockchainUtils.signAndSendTx(tx, identity))
  .then(() => console.log('Successfully transferred ${amount.toNumber()} tokens'))
  .catch(err => {
    console.log('Transfer failed')
  })

    Parameters

    • accountAddressTo: IPublicIdentity["address"]

      Address of the receiver account.

    • amount: BN

      Amount of Units to transfer.

    • Default value exponent: number = -15

      Magnitude of the amount. Default magnitude of Femto-Kilt.

    Returns Promise<SubmittableExtrinsic>

    Promise containing unsigned SubmittableExtrinsic.
