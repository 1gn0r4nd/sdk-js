    Parameters

      The message result of the Claimer's attestation request in requestAttestation.

    • claimer: PublicIdentity

      The PublicIdentity of the claimer. This is also the receiver of the returned message.

    • Default value session: AttesterAttestationSession | null = null

      The AttesterIdentity's session created in initiateAttestation.
