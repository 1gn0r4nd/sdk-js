verify

  • verify(claimHash?: string): Promise<boolean>
    • Defined in src/attestation/Attestation.ts:178

    [ASYNC] Queries an attestation from the chain and checks its validity.

    example
    attestation.verify().then(isVerified => {
  // `isVerified` is true if the attestation is verified, false otherwise
});

    Parameters

    • Default value claimHash: string = this.claimHash

      The hash of the claim that corresponds to the attestation to check. Defaults to the claimHash for the attestation onto which "verify" is called.

    Returns Promise<boolean>

    A promise containing whether the attestation is valid.
