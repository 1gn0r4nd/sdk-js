issuePrivacyEnhancedAttestation

  • issuePrivacyEnhancedAttestation(session: AttesterAttestationSession, reqForAttestation: IRequestForAttestation): Promise<{ attestation: Attestation; witness: Witness }>
    • Defined in src/identity/AttesterIdentity.ts:267

    [ASYNC] Creates a privacy enhanced Attestation for a Claim.

    throws

    ERROR_PE_MISSING.

    Parameters

    • session: AttesterAttestationSession

      The Attester's session which was generated in initiateAttestation.

    • reqForAttestation: IRequestForAttestation

      The Claimer's request for Attestation including all required data to attest the Claim: The Claim itself, the Claimer's signature, the claimHashTree, the cTypeHash, the unique identifier for the delegation, an array of AttestedClaims and the rootHash.

    Returns Promise<{ attestation: Attestation; witness: Witness }>

    A privacy enhanced attestation in array form which represents at index 0 the revocation witness which can be used to revoke the Attestation and at index 1 the Attestation object which should be sent to the Claimer. It can be used to create a Credential.
